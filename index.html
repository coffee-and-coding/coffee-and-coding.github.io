Basic tags
(and what they do)

Tag Name What's It Do?
<!DOCTYPE html> Tells browser what type of document we are giving it
<head> Provides metadata to the browser
<body> Element that contains what we want to give the browser to display
<div> Defines a division or a section in an HTML document
<h1>-<h6> Defines an HTML heading
( 'h' = header | '#' = Level of importance example )
<p> Defines a paragraph section
<a href=" "> Defines a hyperlink to direct you to a different page
( one required attribute: href="" )
<img src=" " alt=" "> Defines an image for the browser to load
( two required attributes: src="" and alt="" )
<ul> or <ol> Defines an unordered list or an ordered list
<li> Defines a list item
( always nested in either <ul> or <ol> tags )
